# Creacion de tipos de datos

Este repositorio contiene ejemplos prácticos relacionados con la creación de tipos de datos en Java. El objetivo es comprender cómo se diseñan tipos de datos abstractos (TDA), cómo se organizan sus componentes internos y cómo se usan desde un cliente, desarrollando habilidades esenciales para POO en Java.

## Objetivo del repositorio

Este repositorio te permitirá dominar la creación y uso de tipos de datos propios, buscando que puedas:

Diseñar clases bien estructuradas y encapsuladas.

Crear constructores seguros y consistentes.

Crear métodos que representen operaciones significativas.

Diferenciar entre tipo de dato y cliente.

Aplicar los tipos de datos para resolver problemas concretos.

## Ejecucion

Identifica en los códigos:

Variables de instancia

Constructores

Métodos

Cliente de prueba

Compila y ejecuta los ejemplos con:

javac NombreClase.java
java NombreClase


Crea tus propios tipos de datos, tomando como guía los ejemplos.

## Actividades recomendadas

Implementar tu propio tipo de dato (ej.: Punto2D, Vector, Segmento).

Modificar los ejemplos para hacer:

nuevos dibujos tortuga

nuevas combinaciones de cargas

nuevas operaciones con complejos

Convertir una clase inmutable en mutable y analizar la diferencia.
